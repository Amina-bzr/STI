Nos Services

Gestion des Switchs

>> Cette application vous offre les informations suivantes :

- L'emplacement géographique du switch (dans quel bloc il se trouve, dans quelle armoire réseau, etc.).

- Les différents VLANs utilisés dans les switchs

- L'adresse MAC du switch

- Le numéro d'inventaire du switch

- Les ports reliés à d'autres équipements (imprimantes, points d'accès, etc.)

- Le type de ports (Gigabits Ethernet, Ethernet)

- Les ports de cascades

Gestion des Vlans

>> Cette application vous permet de :

- Ajouter un nouveau Vlan

- Modifier les informations d'un Vlan

- Retirer Les Vlans non désirés

- Rechercher un Vlan

- Afficher Le nombre de ports pour chaque VLAN
